### Скрипт предназначен для превращения свежеустановленного Arch linux с hyprland в полностью настроенную и готовую к работе систему

```BASH
git clone https://github.com/Trebein/HyprArch
```
```BASH
chmod +X InScript.sh
```
```BASH
sudo -v && ./InScript.sh
```
